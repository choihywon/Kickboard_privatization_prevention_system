# 졸업작품 : 킥보드 사유화 방지 시스템
* 목적 : 뉴스 기사를 통해 사람들이 공용 킥보드를 사유화 한다는 것을 알게됨
* 필요성 : 공용 킥보드의 사유화 방지를 막음
* 기능 :
*        킥보드 사유화를 방지함
*        킥보드가 반납되지 않으면 경고를 함
*       킥보드를 예약할 수 있게 함

*        로그인 페이지와 서버 연결
